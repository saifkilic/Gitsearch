# GitSearch • Developer Discovery

GitSearch is a distraction-free web dashboard designed for exploring GitHub talent, comparing developer profiles side-by-side, and curating your personal collection of inspiring creators. Built with editorial minimalism, massive negative space, and a smooth midnight violet theme[cite: 3, 4].

## Features

- **Advanced Developer Search:** Look up any GitHub username to retrieve rich profile cards detailing bios, organizations, locations, and real-time core metrics like public repositories, followers, and following statistics[cite: 1, 2, 4].
- **Interactive Repository Feed:** Explores up to 12 of a developer's most recently updated repositories with clean, custom-rendered statistical cards detailing star counts and primary languages.
- **Side-by-Side Comparison:** Evaluate two developers simultaneously with a synchronized metric interface to analyze repository and follower traction side-by-side[cite: 1, 2, 4].
- **Instant Bookmarking System:** Save your favorite developers to a curated local collection that updates seamlessly across views without page reloads[cite: 1, 2, 4].
- **Global Trending Discovery:** Discover remarkable open-source talent dynamically curated from high-traction global repositories[cite: 1, 2, 4].
- **Responsive Architecture:** Fully optimized across all screen sizes, from ultra-wide displays down to mobile devices[cite: 3, 4].

## Tech Stack

- **Frontend Architecture:** Semantic HTML5[cite: 1, 4]
- **Style Engine:** Modern CSS3 featuring custom properties, structural backdrop-filters, custom-tuned cubic-bezier animations, and responsive media queries[cite: 3, 4].
- **Reactivity Engine:** Pure Vanilla JavaScript (ES6+) utilizing an encapsulated reactive State-Management architecture with centralized getters and setters.
- **Data Source:** Public GitHub REST API
